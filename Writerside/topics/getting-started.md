# 快速上手

本节旨在为正常智力的人类提供使用指导。

## 先决条件 {#prerequisite}

首先，你需要确认你的设备环境。

- 使用 MacBook Pro 2015 及以上机型
- 搭载 macOS >= 15.7 Sequioa  
- **重要：必须启用SIP (System Integrity Protection)**

如果你在使用过程中遇到任何问题，可以参考 [报告问题](report-an-issue.md) 部分在 GitHub 上提出详细问题。

## 常规使用 {#general}

1. 下载最新的应用商店安装包.
2. 打开终端, 输入 `xattr -cr "/Volumes/xxx/QiuChenAppStore.dmg"` 
3. 挂载 dmg 文件, 并将应用拖入应用程序文件夹.
4. 在启动台启动应用, 并根据提示安装守护进程。
