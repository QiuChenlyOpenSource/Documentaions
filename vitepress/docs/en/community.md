# Community & Messages

## Community Forum {#forum}

### Accessing the Community

Click the **"Community"** icon in the left navigation bar to enter.

The community page displays:
- A list of all app discussion boards
- Hot post previews
- Board search

### App Discussion Boards

Each app automatically creates a dedicated discussion board. Ways to enter:

- Click the app board on the community page
- Click **"Go to Discussion"** on the app detail page

### Browsing Posts

After entering a discussion board, you can see:

- **Post list**: Sorted by latest reply
- **Post cards**: Displaying title, author, time, reply count, like count
- Click a post to enter the detail page

### Creating a Post

1. Click the **"New Post"** button on the discussion board page
2. Fill in the editor dialog:
   - **Title** (required)
   - **Content** (Markdown format supported)
   - **Image attachments** (up to 5 images, 10MB each)
   - **File attachments** (up to 5 files, 20MB each)
3. Click **"Publish"**

### Replies & Interactions

**Replying to posts**:
- Write a reply in the input box at the bottom of the post
- Nested replies supported (up to 3 levels deep)

**Interaction actions**:
- **Like**: Click the like button next to a post/reply
- **Dislike**: Dislike inappropriate replies

**Report**:
- Click the **"Report"** button on a post/reply
- Select report type: Pornography, Fraud, Malicious promotion, Personal attack, Spam, etc.

## User Reviews (Article System) {#articles}

### Browsing Articles

Browse **user review** articles published by users on the community page.

### Writing Articles

1. Go to **Manage** → **My Reviews** → **"Write Review"**
2. Write content in the rich text editor
3. Markdown + Monaco editor supported
4. Can embed app info cards
5. After publishing, can choose public or private

### Article Collections

- Browse curated article collections
- Create your own collections
- Bookmark favorite articles

## Messaging System {#messages}

### Accessing Messages

Click the **"Messages"** icon in the left navigation bar (or enter via the notification icon).

### Conversation List

The left side displays all message conversations, sorted by latest message:

- Recipient's avatar and nickname
- Last message preview
- Unread message count badge

### Sending Messages

1. Click the target contact in the conversation list
2. Enter a message in the input box on the right
3. Press `Enter` to send

### Message Notifications

- A notification badge appears in the top right corner when a new message is received
- Click the **"Notifications"** icon to view the message list

## Notification Center {#notifications}

Click the bell icon in the top right corner to enter the notification page and view:

- Reply notifications
- Like notifications
- Message notifications
- System announcements

## Reporting & Moderation {#report}

If you find rule-violating content:

1. Click **"Report"** in the top right corner of a post/reply
2. Select the report type
3. Fill in supplementary notes (optional)
4. Submit for admin review

### Report Types

| Type | Description |
|------|------|
| Pornography | Adult content |
| Fraud | Fraudulent information |
| Malicious Promotion | Promotion/Advertising |
| Personal Attack | Insults/Threats |
| Spam | Meaningless advertising |
| Other | Other violations |
