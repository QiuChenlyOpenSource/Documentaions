# Settings Center

## Accessing Settings {#access}

Click the **"Manage"** icon in the left navigation bar → **"System Settings"** tab.

> **Note**: Local settings such as appearance, downloads, and proxy **do not require login** to modify. Account profile and community-related features require login to unlock.

## Account Security {#account}

Account management features available after login:

### Basic Information

- **Username**: Displays current username
- **Bound Email**: Email bound / Not bound
- **Login Password**: Change password (requires entering current password for confirmation)
- **"Edit Profile"** button: Modify nickname, email, age

### Device Management

Displays current login device information:

- **Current Device** label
- Number of recognized apps
- Multi-device management

### Dangerous Operations

- **"Delete Account"**: Permanently delete account and all data (posts, replies, comments, articles, messages, likes, uploaded files). Irreversible.
- Requires entering current password for confirmation

## Appearance Settings {#appearance}

| Setting | Options | Description |
|--------|------|------|
| **Appearance** | Light / Dark / Follow System | Real-time switch, no restart needed |
| **Language** | 11 languages available | Real-time switch, takes effect immediately |
| **Splash Animation** | On/Off | Genshin Impact-style WebGL opening animation |
| **Genshin Cursor** | On/Off | Global Genshin Impact-style mouse cursor |

### Supported Languages

Simplified Chinese, Traditional Chinese (Taiwan / Hong Kong / Macau), English, 日本語, 한국어, Français, Deutsch, Español, Русский

The language selector is located in the top right corner of the Manage page and can also be set in the setup wizard.

## Download Settings {#download}

| Setting | Description | Default |
|--------|------|--------|
| **Download Path** | File save directory | `~/Downloads/QiuChenlyAppStore` |
| **Simultaneous Downloads** | Maximum concurrent download tasks | 3 |

## Update Settings {#update}

| Setting | Description |
|--------|------|
| **Auto Check for Updates** | Automatically detect app updates |
| **Update Channel** | Stable only / Include beta |

## Proxy Settings {#proxy}

| Setting | Description |
|--------|------|
| **Enable Proxy** | Toggle, configure HTTP/HTTPS/SOCKS5 proxy |
| **Proxy Address** | Proxy server address and port |
| **Authentication** | Username/Password (if required) |

## Installation Settings {#install}

| Setting | Description |
|--------|------|
| **Auto-install after download** | Automatically perform installation after download completes |
| **Clear extended attributes (xattr)** | Remove quarantine attribute after installation to avoid "damaged" prompts |
| **Auto-inject official version** | After downloading and installing an official app, auto-inject and activate if supported |

## Experimental Features {#experimental}

| Feature | Description |
|------|------|
| **QiuChenly Launchpad** | Takes over `⌘ + Space` and four-finger gestures with a custom launchpad. App auto-restarts after toggle |
| **QiuChenly Proxy** | Daemon built-in netproxy engine, globally takes over system traffic. Takes effect immediately |

## Media Settings {#media}

| Setting | Description |
|--------|------|
| Lyrics page style | Switch between two lyrics display styles |
| Music library path | Manage music scan directories |
| Playback cache | Clear playback cache |
| Hotkeys | Customize global playback hotkeys |

## Cleanup {#cleanup}

| Action | Description |
|------|------|
| **Clear Download Cache** | Delete download temporary files; shows current disk usage and file count |
| **Clear Playback Cache** | Clean up player cache to free disk space |
