# Emby Server

## Supported Version: 4.7.14.0

Download:
[https://github.com/MediaBrowser/Emby.Releases/releases/download/4.7.14.0/embyserver-osx-x64-4.7.14.0.zip](https://github.com/MediaBrowser/Emby.Releases/releases/download/4.7.14.0/embyserver-osx-x64-4.7.14.0.zip)

## Usage

Replace `Emby.Web.dll` at `/Applications/EmbyServer.app/Contents/MacOS/Emby.Web.dll`

Replace `embypremiere.js` at `/Applications/EmbyServer.app/Contents/Resources/dashboard-ui/embypremiere/embypremiere.js`
