# Adobe 中心

Adobe 中心提供 Adobe Creative Cloud 全套件的在线获取和下载。

## 进入 Adobe 中心 {#access}

- **首页进入**：在首页「特色合集」区域点击 **「Adobe 全家桶安装器」** 卡片
- **侧边导航**：在应用页找到 Adobe 入口

## 页面布局 {#layout}

Adobe 中心页面为**左右两栏布局**：

### 左侧操作栏

- **标题**：「Adobe 全家桶」
- **副标题**：「在线获取 Adobe 应用 · 一键清理」

操作区：

| 按钮 | 功能 |
|------|------|
| **一键清理** | 清理 Adobe 残留文件、缓存 |
| **刷新目录** | 重新从 Adobe 官方 API 拉取产品列表 |

分类筛选（点击切换，可查看全部或按类别筛选）：

- 全部分类
- 摄影 (Photography)
- 视频音频 (Video & Audio)
- 设计 (Design)
- 插画 (Illustration)
- 社交媒体 (Social Media)
- AI 生成 (Gen AI)
- PDF 工具 (Acrobat & PDF)
- AR/3D
- 测试版 (Beta Apps)
- 预发布 (Prerelease)

### 右侧产品列表

**搜索栏**：顶部有搜索框，输入关键词实时筛选产品。

**产品卡片**：每个 Adobe 产品显示为一行卡片，包含：

- 产品图标
- 产品名称（如 Photoshop、Illustrator）
- **「在线」** 标签（蓝色）
- 简介描述（tagline）或最新版本号
- 分类标签（彩色小标签，最多显示 3 个）
- **「获取」** 按钮（蓝色）

点击 **「获取」** 按钮进入下载配置对话框。

## 下载配置对话框 {#download-dialog}

点击产品卡片上的 **「获取」** 按钮后，弹出两步配置对话框：

### 第一步：配置 (Configure)

| 配置项 | 选项 | 说明 |
|--------|------|------|
| **版本** | 从下拉列表选择 | 默认选最新版本 |
| **平台** | Apple Silicon (ARM) / Intel (x86_64) | 通常自动匹配当前架构 |
| **语言** | 所有语言 / English / 简体中文 / 日本語 / 한국어 / Français / Deutsch / Español / Italiano 等 | 默认「所有语言」 |

::: tip 平台选择
QiuChenly 应用商店 会自动检测你的 Mac 芯片类型（ARM 或 Intel），默认选中对应平台。你也可以手动切换到另一平台。
:::

### 第二步：确认 (Confirm)

显示完整的下载清单：

- **主程序包**：必选，不可取消
- **依赖组件**：自动选中（如 Adobe Common、CC Library 等）
- **可选组件**：可根据需要勾选或取消

确认无误后点击 **「下载」** 按钮，任务自动加入下载管理器。

## 下载管理 {#download-progress}

下载任务进入 **下载管理** 页面后：

- Adobe 下载以专用卡片样式显示（Adobe 红色主题）
- 显示各组件下载进度
- 下载完成后，在下载管理页面点击 **安装** 按钮执行安装

::: tip 离线安装
如果你已有本地 Adobe 安装包，可以在 **下载管理** 页面点击 **「导入 Adobe 安装包」** 按钮，选择包含安装文件的文件夹，扫描后直接安装，无需重新下载。
:::

## 一键清理 {#cleanup}

点击左侧 **「一键清理」** 按钮，自动清理：

- Adobe 临时安装文件
- CC Library 缓存
- 残留的安装日志

::: warning
清理前请确认没有正在进行的 Adobe 下载任务。
:::

## 文件存储 {#storage}

Adobe 下载文件默认保存到：

> `下载目录/Adobe/`

可在 **管理** → **系统设置** → **下载设置** 中修改下载目录。

## 使用须知 {#notes}

::: warning 重要

1. **无需试用**：Adobe 产品安装后直接注入即可使用，不需要先试用
2. **无需登录账户**：不要登录任何 Adobe 账户
3. **支持所有版本**：包括最新版本和未来版本，通杀所有 Adobe 产品
4. **离线使用**：注入后可以完全离线使用，无需网络连接
5. 神经滤镜等高级功能支持离线安装使用（详见 Telegram 频道）
:::

## 常见问题 {#faq}

### Q: 下载速度慢怎么办？

Adobe 官方 CDN 在国内速度可能较慢，建议：
1. 在 **设置** → **代理** 中配置代理
2. aria2 多线程下载默认启用（最高 64 线程）

### Q: 产品列表加载不出来？

1. 确认 daemon 正在运行（在 **活动监视器** 中搜索 `QiuChenlyDaemon`）
2. 点击 **「刷新目录」** 按钮重新拉取
3. 检查网络连接和代理设置

### Q: 下载完成后在哪安装？

下载完成后的 Adobe 安装包位于下载目录中。在 **下载管理** 页面点击对应 Adobe 任务的 **安装** 按钮执行安装。
