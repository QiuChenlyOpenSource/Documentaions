# 应用管理（注入激活）

应用管理是 QiuChenly 应用商店 的核心功能，可以扫描已安装的 macOS 应用并注入破解。

## 界面概览 {#overview}

进入应用管理有两种方式：

- **首页进入**：点击 QiuChenly 应用商店 首页的 **「可激活应用」** 区域
- **侧边导航**：点击左侧导航栏的 **「应用」** 图标

进入后，你会看到：

- **顶部搜索栏**：可搜索应用名称
- **状态筛选标签**：全部 / 可激活 / 已注入 / 不支持
- **应用卡片列表**：每个应用一张卡片，显示图标、名称、版本、状态
- **批量操作栏**（底部）：全选、批量激活

## 应用状态说明 {#status}

每个应用卡片右下角有一个状态标签：

| 标签 | 含义 | 可操作 |
|------|------|--------|
| **激活**（绿色按钮） | 应用支持注入，尚未激活 | 点击激活 |
| **已注入**（蓝色标签） | 已完成注入破解 | 已可用 |
| **更新**（橙色按钮） | 有新版本可用 | 点击更新 |
| **获取**（灰色按钮） | 尚未安装但可下载 | 点击下载 |
| 无标签 | 不支持或未分类 | — |

## 激活应用（注入） {#activate}

### 标准流程

1. 在应用列表中找到目标 App（可通过顶部搜索栏快速定位）
2. 确认状态显示为绿色的 **「激活」** 按钮
3. 点击 **「激活」** 按钮
4. 在弹出的权限确认对话框中输入 **Mac 登录密码**
5. 等待注入完成（通常 3-10 秒），按钮变为蓝色 **「已注入」**

### 注入过程中发生了什么？

```
1. 检查目标应用版本是否在支持列表中
2. 复制对应的 dylib 动态库到应用目录
3. 修改应用主程序，添加 dylib 加载路径
4. 重新签名应用（ad-hoc 签名）
5. 重置 TCC 权限（如需要）
6. 执行额外 Shell 脚本（如配置中定义）
```

### 注意事项

- **注入前必须关闭目标应用**
- 注入过程中请勿强制退出 QiuChenly 应用商店
- 如果注入失败，请检查应用是否为官方原版（已注入过的应用再次注入可能失败）

## 应用详情页 {#app-detail}

点击任意应用卡片进入详情页，包含三个标签页：

### 概览

- 应用图标、名称、版本号
- **「注入激活」** 按钮（当前操作）
- **「获取应用」** 按钮（下载）
- **「打包 DMG」** 按钮（将已注入应用打包为 dmg）
- **「进入讨论区」** 按钮（跳转到该应用的论坛）
- **「报告问题」** 按钮（提交 Bug 反馈）
- 应用介绍描述
- 右侧信息栏：开发者、最新版本、类别

### 预破解版

某些应用提供 **预破解版**（已打好补丁的版本）：

- 显示 **「预破解版」** 标签
- 点击 **「下载预破解版」** 按钮
- 下载后开箱即用，无需再手动注入

### 官方版本

显示 **「官方版本」** 标签的应用：

- 点击 **「下载官方版」** 按钮下载正版
- 下载完成后需手动注入激活

### 版本历史

查看该应用的历史版本和更新记录。

### 讨论

直接跳转到该应用的论坛讨论板块。

## 筛选与搜索 {#filter}

### 状态筛选

顶部标签栏提供快速筛选：

- **全部**：显示所有已安装应用
- **可激活**：仅显示支持注入的应用（绿色）
- **已注入**：仅显示已完成注入的应用（蓝色）

### 搜索应用

1. 点击顶部搜索框（或按 `⌘ + K` 全局搜索）
2. 输入应用名称
3. 实时显示匹配结果

## 批量操作 {#batch}

底部工具栏提供批量操作：

1. 勾选多个应用左侧的复选框
2. 点击 **「全部激活」** 按钮
3. 输入密码确认
4. 等待逐个注入完成

## 应用页功能区域 {#apps-page}

应用页（`/apps`）是浏览和发现应用的主要入口，分为多个功能区域。

### 页面顶栏

标题栏显示 **「应用」** + 已收录应用总数。右侧有四个快捷入口：

| 按钮 | 跳转 |
|------|------|
| **最近发布** | `/recent-releases` 最新上架应用 |
| **编辑推荐合集** | `/editors-choice` 编辑精选 |
| **所有应用** | `/apps/all` 完整应用目录 |
| **兼容应用** | `/apps/all?filter=activatable` 可激活应用列表 |

### 精选 Hero 卡片（需登录）

两张大卡片并排展示，登录后可见：

**左侧暗色卡片（精选应用）**：
- 显示 **「精选应用」** 标签
- 应用名称、版本号、下载次数
- 应用描述简介
- **「获取应用」** 按钮：下载
- **「了解更多」** 按钮：进入详情

**右侧蓝色卡片（来自正版）**：
- 显示 **「来自正版」** / **「官方认证」** 标签
- 官方版本应用名称和描述
- **「获取应用」** 按钮：直接下载最新版（无需选版本）

未登录时显示登录引导面板，提示 **「登录后查看精选下载」**。

### 来自正版排行榜（需登录）

以排名列表形式展示 Top 6 官方正版应用。每行显示：
- 排名序号（1-6）
- 应用图标和名称
- 版本号、分类
- 点击进入应用详情

顶部 **「查看官方应用」** 可跳转完整列表。
