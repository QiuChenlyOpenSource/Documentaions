# B 站视频下载

支持登录 B 站账号、浏览收藏夹与合集,并把视频解析下载到本地,统一在[下载](./download-manager)页管理;下载完成后可直接用[原生视频播放器](./video-player)播放。

## 登录账号 {#login}

进入 B 站登录页(支持**多账号管理**),通过**扫码登录**:

![b1](/images/bilibili-download/image-1.png)

1. 选择登录方式:**TV 端登录** 或 **Web 端登录**
2. 打开手机上的 B 站客户端 / App / 网页版,点击左上角头像选择 **「扫一扫」**
3. 扫描页面上的二维码,并在手机上确认登录

二维码过期可刷新重试。已登录的账号会出现在 **「我的账号」** 列表,可随时**添加**或**移除**账号;已开通大会员的账号会显示 **大会员标签**。

- **一键导出 / 导入**:可把已登录账号整体导出为文件备份,并在其它设备上导入,免去重新扫码,方便多设备间迁移登录状态。

### 下载专用账号 {#download-account}

可把某个账号(例如大会员账号)设为 **「下载专用账号」**,之后下载视频统一使用它的登录状态 —— 方便用大会员权限下载 4K / 高画质,不必在浏览和下载之间来回切换账号。

- **不设置时**:自动跟随当前正在浏览的账号
- **指定账号失效时**:自动回退到当前浏览账号

## 浏览与下载收藏夹 {#favorites}

登录后进入 B 站用户主页,可看到 **关注 / 粉丝 / 投稿** 等信息(主页支持**搜索**),以及两个标签页:

![b2](/images/bilibili-download/image-2.png)

![b3](/images/bilibili-download/image-3.png)

- **我的收藏夹**:显示各收藏夹的内容数量、创建时间、公开 / 私密状态
- **收藏的合集**:收藏的视频合集

点开任意收藏夹或合集查看视频列表,然后:

- **单个下载**:点视频上的下载按钮,加入下载队列
- **下载全部到本地**:一次性把整个收藏夹 / 合集创建为多个下载任务
- **在线播放**:无需下载,直接用[原生视频播放器](./video-player)在线观看

::: tip 多 P 视频
多 P 视频以**合集卡片**形式呈现,可**按分 P 单独下载**,并展示分 P、时长等信息。卡片封面角标只显示分 P 序号(P1、P2),完整标题在下拉框中查看。
:::

::: warning 批量下载请谨慎
受 B 站 Web API 限制,**「下载全部到本地」** 会在短时间内发起大量解析请求,容易触发账号会话风控,导致请求**集体失败**。建议**按需逐个下载**。
:::

![b3](/images/bilibili-download/image-4.png)

视频经 B 站 API 解析后,统一进入 **「下载」** 页管理。

## 下载设置 {#settings}

在用户主页右上角的 **「下载设置」** 中调整下载参数(作为「下载」「下载全部」的默认值,高清晰度 / 无损音轨需大会员且视频支持,不可用时自动回退到可用的最高档)。**默认全部开启,画质默认自动选择最高**:

![d1](/images/bilibili-download/image-5.png)

| 设置 | 选项 |
|------|------|
| **清晰度** | 自动最高 / 8K 超高清 / 4K 超清 / 1080P60 / 1080P 高码率 / 1080P / 720P60 / 720P / 480P / 360P(默认自动最高) |
| **封装编码** | HEVC / H.265(默认,体积小)、H.264 / AVC(兼容性最好) |
| **合并为 MP4** | 下载后将音视频合并转码为单个 MP4 |
| **保留源文件** | 合并成功后仍保留原始 m4s 分段 |
| **优先 H.265 源** | 解析时优先选择 H.265 编码的源流 |
| **优先 FLAC 音轨** | 若提供则下载无损音频(需大会员) |
| **附加内容** | 弹幕、元数据 JSON、视频封面、UP 主头像 |

::: tip 弹幕会自动抓取并分类保存
下载时会自动抓取**完整弹幕**,并按类型分别保存 —— **普通弹幕**、**高级 / 特殊弹幕**、**互动弹幕**。播放时[原生视频播放器](./video-player)会自动加载这些本地弹幕(含[高级弹幕特效](./video-player#danmaku))。
:::

## 下载过程 {#progress}

下载卡片会显示当前状态(**排队中 / 下载中 / 已完成 / 已取消**)与所处阶段:

> 解析元数据 → 解析视频流 → 下载视频 → 下载音频 → 合并封装 → 转码

下载结果卡片还会展示 **分辨率 / 帧率彩色分级标签** 与 **本地弹幕条数**,下载时间显示完整的年月日时分。

## 下载结果与后续操作 {#result}

下载完成后,结果卡片提供以下操作:

- **播放**:直接用[原生视频播放器](./video-player)观看;下载的文件加入媒体库扫描目录后,也会出现在媒体库中。
- **更新弹幕**:无需重新下载视频,即可刷新已下载视频的弹幕、封面与信息。
- **重新下载**:为已下载视频切换清晰度重新下载;若选择的新清晰度与已有文件完全一致,则跳过下载、仅刷新信息。
