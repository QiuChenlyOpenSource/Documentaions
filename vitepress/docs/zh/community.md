# 社区与私信

## 社区论坛 {#forum}

### 进入社区

在左侧导航栏点击 **「社区」** 图标进入。

社区页面展示：
- 所有应用讨论板块列表
- 热门帖子预览
- 板块搜索

### 应用讨论区

每个应用自动创建专属讨论板块。进入方式：

- 在社区页点击应用板块
- 在应用详情页点击 **「进入讨论区」**

### 浏览帖子

进入讨论区后，可以看到：

- **帖子列表**：按最新回复排序
- **帖子卡片**：显示标题、作者、时间、回复数、点赞数
- 点击帖子进入详情页

### 发帖

1. 在讨论区页面点击 **「发帖」** 按钮
2. 在弹出的编辑器中填写：
   - **标题**（必填）
   - **内容**（支持 Markdown 格式）
   - **图片附件**（最多 5 张，每张 10MB）
   - **文件附件**（最多 5 个，每个 20MB）
3. 点击 **「发布」**

### 回复与互动

**回复帖子**：
- 在帖子底部输入框编写回复
- 支持楼中楼回复（最多 3 层嵌套）

**互动操作**：
- **点赞**：点击帖子/回复旁的点赞按钮
- **点踩**：点踩不合适的回复

**举报**：
- 点击帖子/回复的 **「举报」** 按钮
- 选择举报类型：涉黄、诈骗、恶意引流、人身攻击、垃圾广告等

## 使用心得（文章系统） {#articles}

### 浏览文章

在社区页可浏览用户发布的**使用心得**文章。

### 写文章

1. 进入 **管理** → **我的使用心得** → **「写心得」**
2. 在富文本编辑器中编写内容
3. 支持 Markdown + Monaco 编辑器
4. 可嵌入应用信息卡片
5. 发布后可选择公开或仅自己可见

### 文章合集

- 浏览精选文章合集
- 创建自己的合集
- 收藏喜欢的文章

## 私信系统 {#messages}

### 进入私信

在左侧导航栏点击 **「消息」** 图标（或通知图标进入）。

### 会话列表

左侧显示所有私信会话，按最新消息排序：

- 对方头像和昵称
- 最后一条消息预览
- 未读消息数角标

### 发送消息

1. 在会话列表点击目标联系人
2. 在右侧输入框输入消息
3. 按 `Enter` 发送

### 消息通知

- 收到新消息时右上角显示通知角标
- 点击 **「通知」** 图标查看消息列表

## 通知中心 {#notifications}

点击右上角铃铛图标进入通知页面，查看：

- 回复通知
- 点赞通知
- 私信通知
- 系统公告

## 举报与审核 {#report}

如发现违规内容：

1. 在帖子/回复右上角点击 **「举报」**
2. 选择举报类型
3. 填写补充说明（可选）
4. 提交后由管理员审核处理

### 举报类型

| 类型 | 说明 |
|------|------|
| 涉黄 | 色情内容 |
| 诈骗 | 诈骗信息 |
| 恶意引流 | 推广/广告 |
| 人身攻击 | 辱骂/威胁 |
| 垃圾广告 | 无意义广告 |
| 其他 | 其他违规行为 |
