# 音乐播放器

QiuChenly 应用商店内置全功能本地音乐播放器，支持音乐库管理、播放、歌词显示、动态封面背景。

## 进入音乐播放器 {#access}

在左侧导航栏点击 **「媒体库」** 图标，或从首页 **「音乐」** 卡片进入。

## 界面布局 {#layout}

音乐模块使用**沉浸式全屏布局**（顶部导航栏自动隐藏，鼠标移到顶部唤出）。

### 侧边导航

音乐模块有独立侧边栏：

| 导航项 | 对应页面 |
|--------|---------|
| **首页** | 媒体库首页，重磅热播/为你推荐/音乐模块 |
| **搜索** | 搜索歌曲/艺人/专辑 |
| **音乐** | 音乐库总览（专辑/艺人/曲目） |
| **管理** | 媒体库管理（添加/扫描音乐目录） |

### 媒体库首页

展示以下内容区块：

- **重磅热播**：热播内容推荐
- **为你推荐**：基于播放历史的个性化推荐
- **我的音乐**：快速进入音乐库
- **最近播放**：最近播放的曲目
- **常听**：最常播放的曲目（显示播放次数）
- **猜你喜欢**：智能推荐曲目

## 音乐库 {#library}

音乐库页面（`/media/music`）包含四个标签页：

### 总览

展示艺人、专辑、曲目的统计概览。

### 专辑

- 按专辑封面网格展示
- 排序方式：**最近发行** / **名称 A-Z** / **年份新→旧** / **年份旧→新** / **曲目最多**
- 类型筛选：**全部类型** / **专辑** / **EP** / **单曲** / **合辑**
- 流派筛选：**全部流派** / 具体流派列表
- 点击专辑进入专辑详情页

### 艺人

- 艺人列表（按名称排序）
- 点击艺人进入艺人详情页，查看该艺人的所有专辑和曲目

### 曲目

- 完整的曲目列表（表格视图）
- 列：**标题** / **专辑** / **时长** / **格式**
- 排序：**名称 A-Z** / **时长长→短** / **时长短→长**
- 格式筛选：**全部格式** / FLAC / MP3 / AAC 等
- 点击曲目开始播放

## 播放控制 {#playback}

### 迷你播放条（全局常驻）

在应用底部始终显示迷你播放条，跨页面不中断：

- **封面缩略图**：左侧显示当前播放曲目封面
- **歌曲信息**：曲名 - 艺人
- **播放/暂停** 按钮
- **上一首/下一首** 按钮
- **播放模式切换**（单按钮三态循环）：
  - **顺序播放**：列表播放完后停止
  - **随机播放**：随机选择下一首
  - **单曲循环**：当前曲目循环播放
- **进度条**：可拖拽跳转
- 点击迷你播放条可展开完整播放页面

### 全局快捷键

系统级快捷键，App 在后台也能控制播放。可在 **管理 → 系统设置 → 媒体设置** 中自定义修改。

| 操作 | 默认快捷键 | 修改方法 |
|------|-----------|---------|
| 播放 / 暂停 | `Ctrl + Option + Space` | 点击「修改」后按下新组合键录制 |
| 上一首 | `Ctrl + Option + ←` | 同上 |
| 下一首 | `Ctrl + Option + →` | 同上 |
| 音量 + | `Ctrl + Option + ↑` | 同上 |
| 音量 - | `Ctrl + Option + ↓` | 同上 |

点击 **「修改」** 后，按下键盘组合键即可录制新的快捷键（按 Esc 取消）。系统会自动检测快捷键冲突。

## 歌词页 {#lyrics}

点击迷你播放条可进入全屏歌词页面。

### 歌词功能

- **自动匹配**：从本地 `.lrc` 文件加载歌词
- **逐字高亮**：歌词逐字同步高亮显示
- **双样式**：两种歌词显示样式（在设置中切换）
- **原生解析**：歌词解析在原生层完成，不依赖前端

### 封面动态背景

歌词页背景根据专辑封面自动取色，产生流动渐变动效。

### 音质标记

歌词页上方显示音质标记：

- **母带**（Master）
- **杜比全景声**（Dolby Atmos）

## 菜单栏面板 {#menubar-panel}

点击 macOS 菜单栏的 QiuChenly 应用商店图标，弹出 **「正在播放」** 快捷面板：

- 专辑封面大图
- 歌曲/艺人/专辑信息
- 播放/暂停、上一首/下一首
- 歌词预览（当前句）
- 音量控制

## 状态栏歌词 {#menubar-lyrics}

在屏幕顶部菜单栏显示专辑封面与当前歌词（App 后台也可见）。可在 **媒体设置** 中开启或关闭。

## Touch Bar 歌词条 {#touchbar}

如果 Mac 配备 Touch Bar，可接管 Touch Bar 显示歌词条，切到其他 App 也保持显示。

**显示模式**：
- **关闭**：不接管 TouchBar
- **主区**：歌词占主区，右侧保留系统亮度/音量控制条
- **全宽**：歌词铺满整条 TouchBar

**布局预设**：
- **歌词+进度**：封面 + 逐字歌词 + 可拖进度条
- **纯歌词**：封面 + 逐字歌词（无进度条，歌词更宽）
- **信息+进度**：封面 + 歌名·歌手·专辑 + 进度条
- **纯信息**：封面 + 歌名·歌手·专辑（无进度条）
- **宽歌词**：无封面 + 逐字歌词 + 进度条（歌词区更大）
- **全屏歌词**：无封面 + 纯逐字歌词全宽

## 自动暂停/恢复 {#auto-pause}

- **切到扬声器自动暂停**：系统输出切到内置扬声器（如拔掉耳机/断开蓝牙）时自动暂停播放
- **接入耳机自动恢复播放**：系统输出切到耳机/外接设备时，若当前为暂停则自动继续播放（默认关闭）

## 系统集成 {#system-integration}

QiuChenly 应用商店接入 macOS 系统媒体 API：

- **控制中心**：显示当前播放信息
- **锁屏界面**：显示专辑封面和播放控制
- **AirPods 等耳机**：线控操作
- **Touch Bar**：播放控制

## 播放历史与推荐 {#history}

- **有效播放计数**：只听够 15 秒才算有效播放
- **播放历史**：记录所有播放过的曲目
- **常听**：基于播放次数排序
- **猜你喜欢**：基于播放历史智能推荐
- **播放次数**：每首曲目显示播放次数（「听了 {n} 次」）

## 媒体库管理 {#library-admin}

在音乐侧边栏点击 **「管理」** 进入。

### 添加音乐目录

1. 点击 **「添加目录」**
2. 选择包含音乐文件的文件夹
3. 系统自动扫描：识别音频文件、读取元数据（标题、艺人、专辑、封面）、计算时长

### 存储卡管理

- 查看各音乐目录占用空间
- 移除不需要的目录
- 重新扫描更新曲库

## 支持的音频格式 {#formats}

| 格式 | 扩展名 | 说明 |
|------|--------|------|
| MP3 | `.mp3` | 有损压缩，最通用 |
| FLAC | `.flac` | 无损压缩，推荐 |
| AAC | `.m4a` `.aac` | Apple 格式 |
| ALAC | `.m4a` `.alac` | Apple 无损 |
| WAV | `.wav` | 无压缩 |
| AIFF | `.aiff` `.aif` | Apple 无压缩 |
| OGG | `.ogg` | 开源有损 |
| WMA | `.wma` | Windows 格式 |

## 播放器设置 {#settings}

在 **管理 → 系统设置 → 媒体设置** 中可配置：

- **歌词页样式**：两种风格切换
- **全局快捷键**：自定义播放/暂停、上一首、下一首、音量+、音量- 的快捷键
- **状态栏歌词**：菜单栏常驻歌词开关
- **Touch Bar 歌词条**：显示模式和布局预设
- **自动暂停/恢复**：切换输出设备时的自动行为
