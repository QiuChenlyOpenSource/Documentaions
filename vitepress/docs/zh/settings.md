# 设置中心

## 进入设置 {#access}

在左侧导航栏点击 **「管理」** 图标 → **「系统设置」** 标签页。

> **注意**：界面外观、下载、代理等**本机设置无需登录**即可修改。账号资料、社区相关功能需要登录后解锁。

## 账号安全 {#account}

登录后可访问的账号管理功能：

### 基本信息

- **用户名**：显示当前用户名
- **绑定邮箱**：已绑定邮箱 / 未绑定
- **登录密码**：修改密码（需输入当前密码确认）
- **「编辑资料」** 按钮：修改昵称、邮箱、年龄

### 设备管理

显示当前登录设备信息：

- **当前设备** 标签
- 已识别的应用数量
- 多设备管理

### 危险操作

- **「注销账号」**：永久删除账号及全部数据（帖子、回复、评论、文章、私信、点赞、上传文件），不可恢复
- 需输入当前密码确认

## 界面设置 {#appearance}

| 设置项 | 选项 | 说明 |
|--------|------|------|
| **外观** | 浅色 / 深色 / 跟随系统 | 实时切换，无需重启 |
| **语言** | 11 种语言可选 | 实时切换，即时生效 |
| **启动动画** | 开/关 | 原神风格 WebGL 开场动画 |
| **原神光标** | 开/关 | 全局原神风格鼠标指针 |

### 支持的语言

简体中文、繁體中文（台灣 / 香港 / 澳門）、English、日本語、한국어、Français、Deutsch、Español、Русский

语言选择器位于管理页面右上角，也可在引导页设置。

## 下载设置 {#download}

| 设置项 | 说明 | 默认值 |
|--------|------|--------|
| **下载路径** | 文件保存目录 | `~/Downloads/QiuChenlyAppStore` |
| **同时下载数** | 最大并发下载任务数 | 3 |

## 更新设置 {#update}

| 设置项 | 说明 |
|--------|------|
| **自动检查更新** | 自动检测应用更新 |
| **更新通道** | 仅稳定版 / 包含测试版 |

## 代理设置 {#proxy}

此处配置的是 **下载代理**：仅影响应用商店自身的下载与网络请求。

| 设置项 | 说明 |
|--------|------|
| **启用代理** | 开关 |
| **代理地址** | 代理服务器地址和端口（如 `http://127.0.0.1:7890`），开启后显示 |

::: tip 与「QiuChenly 代理」的区别
本节的下载代理与 [实验性功能](#experimental) 里的 **「启用 QiuChenly 代理」**（全局透明代理引擎）互相独立、互不影响。全局代理的用法见 [网络代理](./network-proxy)。
:::

## 安装设置 {#install}

| 设置项 | 说明 |
|--------|------|
| **下载后自动安装** | 下载完成后自动执行安装 |
| **清除扩展属性 (xattr)** | 安装后移除隔离属性，避免"已损坏"提示 |
| **官方版本自动注入** | 下载官方应用安装后，若受支持则自动注入激活 |

## 实验性功能 {#experimental}

| 功能 | 说明 |
|------|------|
| **QiuChenly 启动台** | 接管 `⌘ + Space` 与四指手势，自研启动台。切换后自动重启应用 |
| **QiuChenly 代理** | daemon 内置 netproxy 引擎，全局接管系统流量。即时生效 |

## 媒体设置 {#media}

音乐 / 媒体相关设置**不在本页**，而在 **媒体库 → 管理 → 「音乐播放」标签页** 中配置：

| 设置项 | 说明 |
|--------|------|
| 歌词样式 | 两种歌词显示风格切换 |
| 全局快捷键 | 总开关 + 播放控制键位自定义 |
| Touch Bar 歌词条 | 显示模式与布局预设 |
| 状态栏与设备 | 菜单栏状态项、状态栏歌词、切换输出设备时自动暂停/恢复 |

音乐库目录的添加 / 扫描 / 移除同样在 **媒体库 → 管理** 的「媒体库 / 存储」标签页中操作。详见[音乐播放器](./music-player#settings)。

## 清理 {#cleanup}

| 操作 | 说明 |
|------|------|
| **清除下载缓存** | 删除下载临时文件，显示当前占用空间和文件数 |
| **清理播放缓存** | 清理播放器缓存，释放磁盘空间 |
