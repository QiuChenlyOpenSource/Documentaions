# 原生视频播放器

媒体库内置视频播放器,用于观看扫描入库的本地视频,也用于 B 站在线视频的播放。支持弹幕、进度条缩略图预览、剧集连播与 Touch Bar 控制。

::: tip 弹幕 ≠ 字幕
播放器内置的是 **B 站弹幕(danmaku)**,即横向飘过画面的评论,并非 `.srt` / `.ass` 字幕。当前不支持外挂/内挂字幕,详见文末「[尚未支持](#limitations)」。
:::
![danmaku](/images/video-player/image.png)


## 进入播放 {#access}

### 媒体库本地视频

1. 在媒体库的 **「管理」** 中添加包含视频的目录并扫描(按扩展名识别,见[支持的视频格式](#formats))。
2. 入库视频会显示封面、分辨率、时长等信息,出现在媒体库首页 / 详情页 / 库页 / 播放历史中。
3. 点击视频即可在播放器窗口中打开;起播会记入一次播放历史。

### 剧集 / 合集

同一合集的多集会作为**播放队列**连续播放,可在 Touch Bar 影片条上切换 **上一集 / 下一集**(见 [剧集连播与 Touch Bar](#episodes))。

### B 站在线视频

从 B 站入口在线播放时使用同一个播放器。通过 [B 站视频下载](./bilibili-download)下载完成的 B 站视频是本地文件,加入媒体库目录扫描后,同样可在此播放。

## 播放控制 {#controls}

播放器底部控制条包含:

| 控件 | 说明 |
|------|------|
| **播放 / 暂停** | 切换播放状态 |
| **后退 / 前进** | 每次 5 秒 |
| **当前时间 / 总时长** | `MM:SS` 显示 |
| **进度条** | 可拖拽跳转;在线播放时显示缓存进度 |
| **静音 / 音量** | 静音按钮 + 音量滑块 |
| **截图** | 保存当前画面 |
| **全屏** | 进入 / 退出全屏 |
| **技术信息** | 叠加显示编码、码率、缓冲等播放信息 |
| **更多菜单(⋯)** | 弹幕、定时关闭、进度条常驻等(见下) |

## 键盘快捷键 {#hotkeys}

| 按键 | 操作 |
|------|------|
| `Space` | 播放 / 暂停 |
| `←` / `→` | 后退 / 前进 5 秒 |
| `↑` / `↓` | 音量增 / 减 |
| `M` | 静音 |
| `F` | 全屏切换 |
| `I` | 技术信息叠层 |

## 弹幕 {#danmaku}

在 **更多菜单(⋯)** 中控制弹幕:

- **显示弹幕**:开 / 关弹幕
- **弹幕字号**:50% / 75% / 100% / 125% / 150% / 200%
- **弹幕字体**:默认、PingFang SC、Hiragino Sans GB、STHeiti、STKaiti、Yuanti SC
- 支持滚动、顶部固定、底部固定等弹幕模式,可调整不透明度、滚动速度与显示区域


## 进度预览与缓存 {#preview}

- **悬停预览**:鼠标悬停进度条时,显示对应时间点的画面缩略图
- **缓存进度**:在线播放时,进度条会标识已缓存、下载中、加载失败的区间,方便判断可流畅拖动的范围

## 截图 {#snapshot}

点击控制条的截图按钮,保存当前画面。

## 更多菜单 {#more-menu}

- **定时关闭**:关闭 / 播完即停 / 5 / 10 / 20 / 30 / 60 / 120 分钟
- **进度条常驻**:让底部进度条始终可见
- **技术信息面板**:查看编码、码率、缓冲等播放信息

## 剧集连播与 Touch Bar {#episodes}

- **连播**:以合集 / 剧集为单位连续播放
- **Touch Bar 影片条**:配备 Touch Bar 的 Mac 可在 Touch Bar 上显示进度并切换 **上一集 / 下一集**,是否可切换会根据当前在队列中的位置自动判断
- 单文件播放时上 / 下一集不可用

## 支持的视频格式 {#formats}

媒体库扫描时识别以下视频文件:

| 格式 | 扩展名 |
|------|--------|
| Matroska | `.mkv` |
| MP4 | `.mp4` `.m4v` |
| QuickTime | `.mov` |
| AVI | `.avi` |
| Windows Media | `.wmv` |
| 传输流 | `.ts` `.m2ts` `.mts` |
| DVD | `.vob` |
| Flash Video | `.flv` |
| WebM | `.webm` |
| RealMedia | `.rmvb` `.rm` |

## 尚未支持 {#limitations}

以下能力当前**未提供**:

- **字幕**:不支持 `.srt` / `.ass` 等外挂或内挂字幕(仅有弹幕)
- **清晰度选择**:无清晰度 / 码率切换
- **倍速播放**:菜单中保留了档位,但当前仅 **1.0×** 正常速度生效
- **画中画 / 循环 / 后台播放 / 进度记忆续播**:暂未实现
- **缩放模式、硬件解码开关、片头片尾跳过**:菜单中为占位项,尚未启用
