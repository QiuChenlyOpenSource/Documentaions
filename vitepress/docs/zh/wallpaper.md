# 桌面动态壁纸（QiuChen 壁纸）

**QiuChen 壁纸** 是一款交互式 **Live2D 桌面动态壁纸**：在系统壁纸之上、桌面图标之下，渲染一个由自研引擎驱动的 Live2D 角色——跟随鼠标转头、头发随光标物理摆动、循环播放待机动作、点击有反应、自动眨眼与呼吸。

> **状态**：持续开发中的独立小工具。

::: tip 自研引擎
Live2D 引擎完全自研，不依赖官方 Cubism SDK。
:::

## 启动与菜单栏 {#launch}

QiuChen 壁纸 是一个 **独立的菜单栏 App**（`QiuChen 壁纸`），独立于应用商店主界面运行。双击运行后：

- 它是 **菜单栏程序**（无 Dock 图标），顶部菜单栏出现「壁纸」图标，所有操作都在这里。
- 角色立即出现在桌面上，**点击会穿透到桌面**（不挡操作图标），并在所有桌面空间（Space）常驻。

菜单栏「壁纸」菜单包含：**模型库… / 快速切换 / 导入模型… / 位置与大小 / 背景 / 表情 / 暂停·继续 / 重新加载 / 设置… / 关于 / 退出**。

## 选择与导入模型 {#models}

内置随 App 分发的 **Hiyori** 模型，开箱即用；也可以导入自己的 Live2D 模型。

### 模型库面板

菜单点击 **「模型库…」** 打开可视化选择面板：网格陈列内置与导入的所有模型，支持 **搜索过滤**、显示体积标签，并且 **每个格子都是该模型的实时预览**（真的在跑 Live2D，低帧率渲染；划出视口的格子自动挂起，避免一次性加载全部模型卡顿）。

### 快速切换与导入

- **快速切换**：菜单里直接列出所有模型，当前模型带勾选，点一下即切换。
- **导入模型…**：选择一个包含 `*.model3.json` 的目录即可导入（`.moc3` + 贴图 + `motion3.json` 动作 + `exp3.json` 表情 + `physics3.json` 物理）。
- **换模型热键 `⌥⌘L`**：一键轮换到下一个模型。

导入的模型存放在 `~/Library/Application Support/QiuChenWallpaper/models/`；**优先级为「用户导入 > 内置」**。

## 交互与动画 {#interaction}

角色每一帧的表现由一整条合成链驱动，效果自然、不机械：

- **待机动作轮播**：播完一段随机换下一段，段与段之间 **交叉淡化**，不硬切。
- **点击反应**：点击角色触发 `TapBody` 动作，与待机动作淡入淡出叠加。
- **随机眨眼 + 呼吸**：眨眼间隔在 2~6 秒之间随机（不再机械固定），配合呼吸起伏。
- **光标跟随**：头、身、眼球会朝鼠标方向转动。
- **头发物理 + 风力**：头发/裙摆按摆链物理摆动；**移动鼠标的速度会转化为风力** 吹动头发。
- **表情叠加**：模型带 `exp3` 表情时，菜单「表情」可切换，按 Add / Multiply / Overwrite 叠加到基础形态上。

## 外观与布局 {#appearance}

### 位置与大小

菜单「位置与大小」提供预设，角色不必占满整屏：**居中·中等 / 居中·大 / 底部居中 / 底部靠右 / 底部靠左**。

### 背景模式

菜单「背景」在三种铺底之间切换：

- **透明（显示桌面）**：只显示角色，背后是你原本的系统壁纸（默认）。
- **深色** / **浅色**：用纯色铺满整屏作为角色背景。

## 多屏与省电 {#multiscreen}

- **多显示器**：每块屏幕各有一个壁纸窗口，热插拔 / 分辨率变化后自动重新铺满。
- **遮挡自动省电**：当窗口被全屏 App 遮挡、切走 Space、显示器息屏，或你手动「暂停」时，**渲染立即停止**；恢复可见或唤醒后自动继续，空闲时不做无谓的 GPU 绘制。

## 渲染与性能 {#rendering}

- **GPU 加速渲染**：画面流畅，动作与物理实时演算。
- **独显优先**：双 GPU 的 Mac 自动使用独立显卡，避免掉帧。
- **模型库预览流畅**：面板里多个模型同时预览也不卡。

## 设置 {#settings}

菜单「设置…」打开设置窗口（深色玻璃质感），包含：

- **通用**：**开机自启**（基于系统 `SMAppService`，macOS 13+）等选项。
- **激活授权**：激活相关面板。

菜单「关于」打开关于窗口。

## 说明与注意事项 {#notes}

- QiuChen 壁纸 是 **独立运行的菜单栏 App**，与应用商店主程序相互独立；关闭壁纸只需在「壁纸」菜单选择 **退出**。
- 目前内容形态专注 **Live2D 动态角色**；静态图片 / 视频 / 网页壁纸尚在规划中。
- 想临时不显示壁纸，用菜单「暂停」即可，不必退出整个 App。
